Pour visualiser en local :
1. Dézipper le dossier.
2. Double-cliquer sur index.html.

Pour GitHub Pages :
1. Déposer tous les fichiers à la racine du repo.
2. Commit + push sur main.
3. Dans GitHub : Settings > Pages > Deploy from branch > main > /root.
4. L’URL sera visible dans Settings > Pages après déploiement.
